***English:***

Code for managing finances with PySimpleGUI using text inputs for description and amount, displaying the history in a data table. If the user clicks add, the system validates the data against empty fields or negative numbers, calculates the balance in real-time, and updates the summary cards on the screen.

Requirements: To run this project, you need to have the PySimpleGUI library installed. You can install it using pip:

pip install PySimpleGUI

***Português:***

Código para gerenciar finanças com PySimpleGUI usando inputs de texto para descrição e valor, exibindo o histórico em uma tabela. Se o usuário clicar em adicionar, o sistema valida os dados contra campos vazios ou números negativos, calcula o saldo em tempo real e atualiza os cards de resumo na tela.

Requisitos: Para rodar este projeto, é necessário ter a biblioteca PySimpleGUI instalada. Você pode instalá-la usando o pip:
